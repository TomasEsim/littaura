# Littaura — Didmeninė kuro prekyba

Statinis landing puslapis didmeninei kuro prekybai visoje Lietuvoje.

## Funkcijos
- Responsyvus dizainas (mobile / tablet / desktop)
- Interaktyvi kuro kainų lentelė su gyvai skaičiuojama suma
- Kuro rūšies pasirinkimas (DK / DKK / DKU)

## Paleidimas
Tai statinis puslapis — atidaryk `index.html` naršyklėje arba paleisk:

```bash
python3 -m http.server 4173
```

ir atidaryk http://localhost:4173

## Struktūra
- `index.html` — puslapio struktūra
- `styles.css` — stiliai ir responsyvumas
- `script.js` — kuro pasirinkimas ir kainos skaičiavimas
